HTML Contagem Regressiva.

Escrito por Danilo Yuji Okazaki Publicado em: 29 de janeiro de 2024 | Atualizado em: 29 de janeiro de 2024

crie um código em html para contagem regressiva

Aqui está um exemplo simples de como criar uma contagem regressiva em HTML usando JavaScript:

```



Contagem Regressiva



Contagem regressiva de 5 minutos


Tempo restante:




```

Esse código usa a variável `tempoRestante` para armazenar o número de segundos restantes na contagem regressiva. A função `contagemRegressiva` é chamada a cada segundo usando `setInterval` e atualiza o contador na página HTML. Quando o tempo se esgota, a função limpa o intervalo e exibe uma mensagem.

crie um código em html que faça o preço mudar em 10 segundos após a pessoa passar o mouse por um botão

Aqui está um exemplo de código HTML que faz o preço mudar em 10 segundos após o usuário passar o mouse por um botão:

```



Preço Dinâmico



Produto XYZ


Preço: R$100.00





```

Este código usa a função JavaScript `setTimeout` para aguardar 10 segundos antes de chamar a função `mudarPreco`. Quando o usuário passa o mouse sobre o botão, o temporizador é iniciado e, após 10 segundos, o preço do produto é reduzido em 10% e atualizado na página. Note que o preço é arredondado para duas casas decimais usando a função `toFixed`.


crie um código em html que faça o preço descontar gradativamente até um determinado valor após a pessoa clicar em um botão

Aqui está um exemplo de código HTML que faz o preço de um produto descontar gradativamente até um determinado valor após o usuário clicar em um botão:

```



Preço Gradativo



Produto XYZ


Preço: R$100.00





```

Este código usa a função JavaScript `setTimeout` para atualizar o preço do produto a cada segundo, diminuindo o valor gradativamente até atingir o preço mínimo definido na variável `precoMinimo`. Quando o usuário clica no botão, a função `atualizarPreco` é chamada e o preço começa a diminuir gradativamente. O desconto é calculado com base na variável `taxaDesconto`, que é definida como 5% no exemplo. O preço é arredondado para duas casas decimais usando a função `toFixed`.

Aqui está um exemplo de código HTML que faz o preço de um produto descontar gradativamente até um determinado valor após o usuário clicar em um botão e, após o desconto, aparece um botão de compra:

```



Preço Gradativo com Botão de Compra



Produto XYZ


Preço: R$100.00






```

Este código é semelhante ao exemplo anterior, mas adiciona um botão de compra que é exibido quando o preço atinge o valor mínimo. A função `atualizarPreco` é chamada quando o usuário clica no botão "Clique para obter desconto gradativo", diminuindo o preço gradativamente até atingir o preço mínimo definido na variável `precoMinimo`. Quando o preço mínimo é atingido, o botão de compra é exibido usando a propriedade `display` do CSS. O botão de compra é oculto por padrão usando a propriedade `display: none`. O desconto é calculado com base na variável `taxaDesconto`, que é definida como 5% no exemplo. O preço é arredondado para duas casas decimais usando a função `toFixed`.

Quando o preço mínimo é atingido, o botão de compra é exibido e o botão de desconto é ocultado. O desconto é calculado com base na variável `taxaDesconto`, que é definida como 5% no exemplo.
