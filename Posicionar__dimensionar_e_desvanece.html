Para redimensionar cada imagem e posicioná-las em uma posição específica, você pode modificar o código CSS e adicionar algumas propriedades. Aqui está uma versão atualizada do código, onde cada imagem é redimensionada e posicionada conforme solicitado:

```html

integrity="sha384-9ndCyUaIbzAi2FUVXJi0CjmCapSmO7SnpJef0486qhLnuZ2cdeRhO02iuK6FUUVM" crossorigin="anonymous">

```

Nesse código, adicionamos as seguintes propriedades CSS ao seletor `ul li div img`:

- `max-width: 100%;` e `max-height: 100%;` redimensionam a imagem para ajustá-la ao contêiner mantendo a proporção original.

- `position: absolute;` posiciona a imagem de forma absoluta dentro do contêiner.

- `top: 50%;` e `left: 50%;` posicionam a imagem verticalmente e horizontalmente no centro do contêiner.

- `transform: translate(-50%, -50%);` centraliza a imagem corretamente ajustando sua posição em relação ao seu próprio tamanho.

Essas alterações permitirão que você redimensione cada imagem e as posicione como desejado dentro do contêiner. Lembre-se de ajustar o valor da variável `--divHeight` para a altura desejada para cada imagem. Certifique-se de fornecer os caminhos corretos para as imagens no atributo `src` das tags ``.

    Neste exemplo, adicionamos a variável `--sensitivity` ao CSS com um valor padrão de `0.5`. Esse valor pode ser ajustado para controlar a sensibilidade do surgimento das imagens.

    No JavaScript, usamos a variável `sensitivity` para obter o valor definido no CSS e multiplicamos o fator de sensibilidade pelo índice da imagem (aumentando o fator para cada imagem subsequente). Isso permite ajustar a sensibilidade individualmente para cada imagem.

    Em seguida, usamos a fórmula de cálculo da opacidade levando em consideração o fator de sensibilidade e a posição de rolagem. Aumentar o fator de sensibilidade torna as imagens mais sensíveis à rolagem e aumenta a velocidade de surgimento.

    Experimente ajustar o valor da variável `--sensitivity` no CSS para aumentar ou diminuir a sensibilidade das imagens em relação à rolagem.
