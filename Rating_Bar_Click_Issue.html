Rating Bar Click Issue

Escrito por Danilo Yuji Okazaki Publicado em: 31 de janeiro de 2024 | Atualizado em: 31 de janeiro de 2024

porque o código abaixo, quando clico na barra de progresso não esta aparecendo o respectivo rating?

O código parece estar correto, mas aqui estão algumas coisas que você pode verificar para descobrir o problema:

1. O problema pode estar na função `showReviews(rating)`. O código que você forneceu não contém essa função, então eu não posso analisá-la. Verifique se essa função existe e está funcionando corretamente.

2. Verifique se os elementos da página HTML estão realmente recebendo o atributo `data-rating` conforme esperado. Você pode fazer isso inspecionando o elemento no navegador.

3. O código JavaScript que você forneceu está utilizando a função `matches` para verificar se o elemento clicado tem a classe `.rating-link`. No entanto, este método verifica se o elemento se encaixa no seletor CSS fornecido. Portanto, se o clique ocorrer em um elemento filho do elemento com a classe `.rating-link`, esse método retornará `false`.

Para contornar esse problema, você pode usar o método `closest` em vez de `matches`. Isso verificará o elemento clicado e seus ancestrais para ver se algum deles se encaixa no seletor CSS fornecido. Aqui está como você pode alterar seu código para usar `closest`:

```javascript

document.addEventListener('click', function (e) {

var target = e.target.closest('.rating-link');

if (target) {

e.preventDefault();

var rating = parseInt(target.getAttribute('data-rating'));

showReviews(rating);

}

});

```

4. O uso de aspas duplas ao redor do atributo `style` na linha que cria o elemento `progress` parece estar incorreto (`"height: 25px"`). Isso pode ser um erro de digitação e você pode precisar corrigi-lo para `'height: 25px'`.

Por favor, revise estas sugestões e veja se elas ajudam a resolver o problema. Se o problema persistir, por favor, forneça mais informações para que eu possa te ajudar melhor.

const slot1 = document.getElementById('slot1');

const slot2 = document.getElementById('slot2');

const slot3 = document.getElementById('slot3');

const spinButton = document.getElementById('spin-button');

const discountButton = document.getElementById('discount-button');

const result = document.getElementById('result');

const countdownd = document.getElementById('countdownd');

//let attempts = 0;

let attempts = localStorage.getItem('attempts') ? Number(localStorage.getItem('attempts')) : 0;

spinButton.addEventListener('click', () => {

if (attempts < 3) {

attempts++;

let spinning1 = setInterval(() => {

slot1.innerText = emojis[Math.floor(Math.random() * emojis.length)];

}, 100);

let spinning2 = setInterval(() => {

slot2.innerText = emojis[Math.floor(Math.random() * emojis.length)];

}, 100);

let spinning3 = setInterval(() => {

slot3.innerText = emojis[Math.floor(Math.random() * emojis.length)];

}, 100);

setTimeout(() => {

clearInterval(spinning1);

clearInterval(spinning2);

clearInterval(spinning3);

if (attempts === 3) {

slot1.innerText = '🍀';

slot2.innerText = '🍀';

slot3.innerText = '🍀';

result.innerText = 'Congratulazioni! Hai appena ricevuto uno sconto!';

result.classList.add('blinking');

result.style.color = 'green';

discountButton.style.display = 'block';

countdownd.style.display = 'block';

let timeLeftd = 30;

countdownd.innerText = `Lo sconto scade il: ${timeLeftd} secondi`;

let countdowndInterval = setInterval(() => {

timeLeftd--;

countdownd.innerText = `Lo sconto scade il: ${timeLeftd} secondi`;

if (timeLeftd <= 0) {

clearInterval(countdowndInterval);

discountButton.style.display = 'none';

countdownd.style.display = 'none';

result.style.display = 'none';

}

}, 1000);

} else {

result.innerText = 'Purtroppo non hai vinto. Riprova.';

discountButton.style.display = 'none';

result.style.color = 'red';

}

if (attempts >= 3) {

resultElement.textContent = 'Hai già speso tutti i tuoi tentativi. Il tuo sconto è già stato concesso. Grazie per aver partecipato!';

spinButton.disabled = true;

}

}, 1000);

}

});

O código que você forneceu parece ser parte de uma máquina caça-níqueis simples que usa emojis em vez de números ou imagens. Ele permite ao usuário "girar" a máquina até três vezes, e parece oferecer um desconto se o usuário acertar três "🍀" após a terceira tentativa.

O botão de giro (`spinButton`) é desabilitado após três tentativas. Portanto, sim, o botão de giro será desabilitado após três tentativas, como sugerido pela linha:

```javascript

if (attempts >= 3) {

resultElement.textContent = 'Hai già speso tutti i tuoi tentativi. Il tuo sconto è già stato concesso. Grazie per aver partecipato!';

spinButton.disabled = true;

}

```

No entanto, há um pequeno problema com o código. `resultElement` não é definido em nenhum lugar do código que você forneceu. Você provavelmente queria usar `result` em vez disso, como está usando em outras partes do código. Aqui está como você pode corrigir essa linha:

```javascript

if (attempts >= 3) {

result.textContent = 'Hai già speso tutti i tuoi tentativi. Il tuo sconto è già stato concesso. Grazie per aver partecipato!';

spinButton.disabled = true;

}

```

Também vale a pena notar que, embora você esteja tentando armazenar a quantidade de tentativas no `localStorage`, você não está atualizando esse valor no `localStorage` após cada tentativa. Aqui está como você pode atualizar o valor após cada tentativa:

```javascript

spinButton.addEventListener('click', () => {

if (attempts < 3) {

attempts++;

localStorage.setItem('attempts', attempts.toString());

// ...

```

Isso garantirá que o número de tentativas seja lembrado mesmo se o usuário recarregar a página.

quero que caso o usuário tente atualizar a pagina, não consiga mais tentar ganhar o desconto

O seu código atual já guarda o número de tentativas no localStorage, o que é mantido mesmo que a página seja atualizada. Isso significa que se o usuário já tentou 3 vezes e atualiza a página, ele não poderá tentar novamente.

No entanto, para impedir que o usuário tente novamente após atualizar a página, você deve desabilitar o botão de rotação assim que a página for carregada se o número de tentativas já for 3. Aqui está como você pode fazer isso:

```javascript

window.onload = function() {

if (attempts >= 3) {

spinButton.disabled = true;

result.textContent = 'Hai già speso tutti i tuoi tentativi. Il tuo sconto è già stato concesso. Grazie per aver partecipato!';

}

}

```

Agora, se o número de tentativas for 3 ou mais quando a página for carregada, o botão de rotação será desabilitado imediatamente. Isso ocorrerá mesmo que o usuário atualize a página, porque o número de tentativas é armazenado no localStorage e não é perdido quando a página é atualizada.
