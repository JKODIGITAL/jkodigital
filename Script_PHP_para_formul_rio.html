Para mostrar a mensagem de sucesso e depois redirecionar o usuário de volta para a página do formulário, você pode usar um pouco de JavaScript junto com o PHP. Vou modificar o script `processa_formulario.php` para incluir essa funcionalidade.

Neste script, após o processo de envio do e-mail, uma caixa de alerta JavaScript é exibida com a mensagem de sucesso ou erro. Em seguida, a função `window.location.href` redireciona o usuário de volta para a página do formulário. Substitua `/sua-pagina-de-formulario.html` pelo caminho da sua página de formulário.

Notas Importantes:

- Certifique-se de que a página do formulário está corretamente especificada no `window.location.href`.

- Lembre-se de adicionar o atributo `name` a todos os campos do formulário para que possam ser capturados pelo script PHP.

- Realize as validações necessárias dos dados no script PHP para evitar problemas de segurança, como injeções de código.

se eu quiser mandar para varios e-mails

Para enviar o formulário para vários e-mails, você pode modificar o script PHP para incluir um array com os endereços de e-mail desejados. Vou mostrar como você pode fazer isso:

Script PHP Modificado para Vários Destinatários (processa_formulario.php)

```php

if ($SERVER["REQUESTMETHOD"] == "POST") {

$nome = $_POST['name'] ?? '';

$idade = $_POST['idade'] ?? '';

$data = $_POST['data'] ?? '';

$cep = $_POST['cep'] ?? '';

$email = $_POST['email'] ?? '';

$mensagem = $_POST['message'] ?? '';

// Validar os dados aqui

// Lista de destinatários

$destinatarios = ['destinatario1@exemplo.com', 'destinatario2@exemplo.com']; // Adicione os endereços de e-mail aqui

$subject = 'Nova Mensagem do Formulário de Contato';

$message = "Nome: $nome\nIdade/Tamanho: $idade\nData da Ocasião: $data\nCEP: $cep\nE-mail: $email\nMensagem: $mensagem";

$headers = "From: webmaster@seusite.com.br\r\n";

$headers .= "Reply-To: $email\r\n";

$envioSucesso = true;

// Enviar o e-mail para cada destinatário

foreach ($destinatarios as $to) {

if(!mail($to, $subject, $message, $headers)) {

$envioSucesso = false;

break;

}

}

// Verificar se todos os e-mails foram enviados com sucesso

if ($envioSucesso) {

echo "";

} else {

echo "";

}

}

?>

```

Explicação do Código:

- Array de Destinatários: Um array `$destinatarios` é criado contendo todos os endereços de e-mail para os quais você deseja enviar a mensagem.

- Loop de Envio: O script percorre o array de destinatários, enviando o e-mail para cada um. Se algum dos e-mails falhar, a variável `$envioSucesso` se torna `false`.

- Verificação de Sucesso: Após tentar enviar para todos os destinatários, o script verifica se todos os e-mails foram enviados com sucesso e exibe uma mensagem apropriada.

Notas Importantes:

- Certifique-se de que os endereços de e-mail estão corretos.

- Lembre-se de adicionar o atributo `name` a todos os campos do formulário.

- É importante realizar as validações dos dados recebidos para evitar problemas de segurança.

CLIQUE PARA VER MAIS VESTIDOS...

Para tornar mais evidente que as imagens são clicáveis e levam ao blog sobre o tema, você pode aplicar algumas técnicas visuais e de interação. Aqui estão algumas sugestões:

1. Efeito de Hover nas Imagens

Adicionar um efeito visual quando o mouse passa sobre a imagem pode indicar que ela é interativa. Por exemplo, você pode adicionar um leve efeito de zoom ou mudar a opacidade da imagem.

CSS para Efeito de Hover

```css

.collection-item img {

transition: transform 0.3s ease, opacity 0.3s ease;

}

.collection-item img:hover {

transform: scale(1.05);

opacity: 0.8;

}

```

2. Ícone de Link ou Botão "Saiba Mais"

Adicionar um ícone de link ou um botão "Saiba Mais" sobre a imagem também pode indicar interatividade. Isso pode ser feito com um pequeno ícone de seta ou lupa que aparece no meio da imagem quando o mouse passa sobre ela.

4. Animação ou Efeito no Cursor

Um efeito adicional pode ser mudar o estilo do cursor para `cursor: pointer;` ao passar sobre a imagem, reforçando a ideia de que é um elemento clicável.

CSS para Cursor


Sim, você pode usar o Popper (uma biblioteca popular para tooltips e popovers) em um site para computadores e criar alternativas intuitivas para smartphones. Vou detalhar como você pode fazer isso para ambos os cenários:

Popper.js é uma biblioteca que permite criar tooltips e popovers que podem ser usados para indicar que uma imagem é clicável.

2. Alternativas Intuitivas para Smartphones

Em smartphones, onde o hover não é uma opção, você pode usar diferentes métodos para indicar que uma imagem é clicável:

- Ícones Visíveis: Adicione um ícone de link ou uma lupa diretamente sobre a imagem. Isso pode ser feito com um ícone de "seta" ou "mais" que fica visível o tempo todo.

```html

```

- Bordas ou Sombras ao Redor da Imagem: Estilizar as imagens com bordas ou sombras destacadas pode também indicar que são elementos interativos.

```css

```

- Animações Subtis: Adicionar uma animação sutil que é ativada ao carregar a página, como um breve "pulsar" da imagem, pode chamar atenção para a interatividade da imagem.

```css

0% { transform: scale(1); }

50% { transform: scale(1.05); }

100% { transform: scale(1); }

}

.collection-item img {

animation: pulse 2s infinite;

}

```

Conclusão

Para computadores, o uso de popovers ou tooltips do Popper.js pode ser uma solução eficaz. Para smartphones, onde o hover não é uma opção, ícones visíveis, bordas destacadas, sombras, ou animações sutis podem ser usadas para indicar que a imagem é clicável. Lembre-se de testar a funcionalidade em vários dispositivos para garantir uma experiência de usuário consistente.
