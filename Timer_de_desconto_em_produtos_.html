Timer de desconto em produtos.

Escrito por Danilo Yuji Okazaki Publicado em: 31 de janeiro de 2024 | Atualizado em: 31 de janeiro de 2024

crie um código de html de um site com uma grade de produtos, onde cada produto tem um timer de desconto de 20 segundos que vai se expirando e quando o timer acaba o botão de compra desaparece.

Para criar um site com uma grade de produtos e um timer de desconto de 20 segundos, você pode usar HTML, CSS e JavaScript. Abaixo está um exemplo de como criar a estrutura básica do site com esses recursos.

1. Crie um arquivo chamado `index.html` e cole o seguinte código:

```html

Loja com Timer de Desconto

Nome do Produto 1

Descrição do Produto 1

```

2. Crie um arquivo chamado `styles.css` e cole o seguinte código:

```css

body {

font-family: Arial, sans-serif;

margin: 0;

padding: 0;

}

.product-grid {

display: flex;

flex-wrap: wrap;

justify-content: center;

}

.product {

box-sizing: border-box;

padding: 16px;

width: 300px;

margin: 16px;

border: 1px solid #ccc;

text-align: center;

}

.timer {

font-size: 1.2em;

font-weight: bold;

margin-bottom: 8px;

}

.buy-button {

background-color: #007bff;

border: none;

color: white;

padding: 8px 16px;

text-align: center;

text-decoration: none;

display: inline-block;

font-size: 16px;

margin: 4px 2px;

cursor: pointer;

}

```

3. Crie um arquivo chamado `scripts.js` e cole o seguinte código:

```javascript

const products = document.querySelectorAll('.product');

const discountDuration = 20000; // 20 segundos

function startTimer(product) {

const timerElement = product.querySelector('.timer');

const buttonElement = product.querySelector('.buy-button');

let timeRemaining = discountDuration / 1000;

timerElement.textContent = `${timeRemaining}s`;

const intervalId = setInterval(() => {

timeRemaining--;

timerElement.textContent = `${timeRemaining}s`;

if (timeRemaining <= 0) {

clearInterval(intervalId);

timerElement.textContent = 'Desconto expirado';

buttonElement.style.display = 'none';

}

}, 1000);

}

products.forEach((product) => {

startTimer(product);

});

```

Esse código cria uma página HTML com uma grade de produtos, onde cada produto tem um timer de desconto de 20 segundos. Quando o timer chega a 0, o botão de compra desaparece. Lembre-se de substituir os nomes e descrições dos produtos, bem como as URLs das imagens, pelos seus próprios dados.
