Vou listar algumas correções necessárias para a barra lateral (sidebar) funcionar corretamente com a função toggle e seja responsiva em telas de smartphone:

1. Parece que você está usando Bootstrap 5.3.0, que tem um componente chamado Offcanvas. Você tentou utilizar este componente adicionando as classes "offcanvas" e "offcanvas-start" na div da barra lateral. Para o funcionamento correto desse componente, você deve adicionar o script do Bootstrap JS em sua página.

2. Você possui três botões com o id "sidebarToggle" na página. O id de um elemento deve ser único em toda a página. Por isso, você deveria remover os dois extras.

3. Em Bootstrap, o offcanvas pode ser acionado por um botão com o atributo `data-bs-toggle="offcanvas"`. Portanto, ao invés de criar um evento de clique, você pode usar este atributo.

    preciso que ao clicar em qualquer link do menu, ele feche o menu

    No Bootstrap 5, é possível fazer com que o offcanvas feche ao clicar em um link usando JavaScript/jQuery. Aqui está um exemplo de como você pode implementar isso:

    ```javascript

    // jQuery

    $(document).ready(function () {

    $('#sidebar a').on('click', function () {

    var bsOffcanvas = new bootstrap.Offcanvas(document.getElementById('sidebar'));

    bsOffcanvas.hide();

    });

    });

    ```

    ```javascript

    // JavaScript Puro

    document.addEventListener('DOMContentLoaded', function () {

    var sidebarLinks = document.querySelectorAll('#sidebar a');

    sidebarLinks.forEach(function (link) {

    link.addEventListener('click', function () {

    var sidebar = document.getElementById('sidebar');

    var bsOffcanvas = new bootstrap.Offcanvas(sidebar);

    bsOffcanvas.hide();

    });

    });

    });

    ```

    Este código adiciona um ouvinte de evento 'click' a todos os links dentro do elemento com o ID `#sidebar`. Quando um link é clicado, ele chama a função que esconde o offcanvas.

    Note que `bootstrap.Offcanvas` é uma classe do Bootstrap 5 usada para controlar o comportamento do offcanvas. Nós criamos uma nova instância dessa classe para o elemento `#sidebar` e chamamos o método `hide()` para esconder o offcanvas.

    Certifique-se de adicionar este script depois de carregar o JavaScript do Bootstrap no seu HTML.
