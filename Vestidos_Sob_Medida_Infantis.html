2. CSS para o Efeito Shine:

Aqui está o CSS que adiciona o efeito "shine":

```css

.image-container {

position: relative;

overflow: hidden; / Garante que o brilho não ultrapasse os limites da imagem /

}

.image-container img {

display: block; / Remove espaçamentos desnecessários /

width: 100%; / Ajuste conforme necessário /

height: auto; / Mantém a proporção da imagem /

transition: transform 0.3s ease; / Efeito suave ao passar o mouse /

}

.image-container::after {

content: '';

position: absolute;

top: -50%;

left: -50%;

width: 200%;

height: 200%;

background: linear-gradient(to right, transparent, rgba(255, 255, 255, 0.8), transparent);

transform: rotate(30deg);

transition: all 0.7s ease; / Tempo de transição do efeito brilho /

pointer-events: none; / Evita interferência com interações do mouse /

}

.image-container:hover::after {

left: 150%; / Move o brilho através da imagem /

}

Neste código, o pseudo-elemento `::after` cria uma camada sobre a imagem com um gradiente que simula o brilho. Ao passar o mouse sobre a imagem, este gradiente se move, criando o efeito "shine".

Para configurar uma imagem de fundo diferente na `.hero-section` para telas menores, você pode usar a funcionalidade de media queries do CSS. Com media queries, é possível especificar estilos diferentes para diferentes tamanhos de tela. Vou demonstrar como ajustar sua classe `.hero-section` para usar uma imagem de fundo diferente em dispositivos com telas menores.

Exemplo de Media Query para Telas Menores:

Você pode definir um ponto de quebra (breakpoint) para telas menores, como 768px, que é um tamanho comum para tablets. Abaixo desse tamanho, uma imagem de fundo diferente será aplicada.

CSS:

```css

.hero-section {

background: url('/assets/images/banner1.webp') no-repeat center center;

background-size: cover;

padding: 100px 50px;

height: 70vh;

}

/ Media Query para telas menores (por exemplo, abaixo de 768px) /

@media (max-width: 768px) {

.hero-section {

background: url('/assets/images/banner-small.webp') no-repeat center center;

background-size: cover;

}

}

```

Explicação:

- A regra `.hero-section` define o estilo padrão para a seção, incluindo a imagem de fundo para telas maiores.

- A media query `@media (max-width: 768px)` aplica-se quando a largura da tela é igual ou menor que 768 pixels. Dentro desta media query, a regra `.hero-section` é sobrescrita com uma nova imagem de fundo, que é mais adequada para telas menores.

Considerações Adicionais:

- Teste em Diferentes Dispositivos: É importante testar essa configuração em diferentes dispositivos para garantir que as imagens de fundo apareçam corretamente em todas as resoluções de tela.

- Otimização de Imagens: Certifique-se de que as imagens de fundo estejam otimizadas para a web, para que não afetem negativamente os tempos de carregamento da página, especialmente em dispositivos móveis.
